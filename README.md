# SQL Data Testing Portfolio – End-to-End BI Validation

Complete data validation project using public datasets (simulating CRM + ERP sources).

## What I built
- Executed complex SQL queries to verify data accuracy between **source**, **staging** and **final report** layers
- Created full test documentation with business requirements mapping and defect examples
- Directly matches Inetum’s need for SQL proficiency and report correctness

## Technologies
SQL · Jupyter Notebook · Pandas · SQLite · Excel

## Project Structure
- `data/source/` → raw CRM + ERP
- `sql/` → layer transformations + 30+ validation queries
- `notebooks/end_to_end_validation.ipynb` → full pipeline execution
- `docs/` → business requirements + test cases + defects log

## How to run
1. `git clone https://github.com/GiorgioF28/sql-bi-testing-portfolio.git`
2. Apri `notebooks/end_to_end_validation.ipynb`
3. Run all cells
4. Controlla `reports/final_report_example.csv` e defects

Perfetto per ruoli QA Analyst / BI Tester.
